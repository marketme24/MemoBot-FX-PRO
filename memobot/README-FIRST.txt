MEMOBOT - ONE CLICK LAUNCH
==========================

To run this platform locally without any terminals or coding:

PREREQUISITES:
1. You must have Node.js installed on your computer. If you don't have it, download and install it from https://nodejs.org/ (The LTS version is recommended).

HOW TO START:

WINDOWS USERS:
- Simply double-click on the "Launch-Windows.vbs" file.
- It will silently install everything needed and automatically open the bot in your default web browser after a few seconds. 
- You won't see any black terminal windows!

MAC USERS:
- Double-click on "Launch-Mac.command".
- It might ask for permissions the first time. If it does not run, you may need to open Spotlight (Cmd+Space), type "Terminal", drag the Launch-Mac.command file into it, and hit Enter.

TO STOP THE BOT:
- Windows: Open Task Manager, look for "node.js" processes, and end them.
- Mac: In Activity Monitor, force quit the "node" process.
